🛒 Blinkit Clone – Grocery Delivery Web App

A Blinkit Clone built using Spring Boot (Java) and basic Web technologies that simulates a simple grocery delivery platform. The application allows users to view products, interact with a backend API, and manage product data through RESTful services.

This project demonstrates full-stack development concepts, including frontend integration, REST API design, backend architecture, and database interaction.

🚀 Features

📦 Display grocery products
➕ Add new products via API
🔎 Search products
🔗 Frontend connected with backend APIs
⚡ RESTful API using Spring Boo
🧪 API testing using Postman
